How to Use BookMySpecialist

Your guide to booking and managing appointments easily

Booking an Appointment
  • Click Book Now next to your desired specialty.
  • Choose a doctor and available time slot from the dropdown.
  • After clicking Book Now, a form will appear. Enter your Full Name, Email, and Phone Number.
  • Click Confirm to submit your booking.
  • A confirmation message with your appointment details will pop up.
Tip: You must select a doctor before you can proceed with booking.
Navigating the Website
  • Use the About us button to learn more about BookMySpecialist.
  • The Help button (this page) is always available at the top for guidance.
  • Click Admin to access administrative tools (password required).
  • To return to the main booking page, use the Back to Main Page link below.
Managing Appointments
  • Each appointment is assigned a unique ID automatically.
  • If you delete all appointments, the next appointment ID may not start from 1 (this is normal database behavior).
  • Admins can view and manage all appointments from the admin panel.
Note: For your privacy, always double-check your contact details before confirming your booking.
For Administrators
  • Click the Admin button and enter the correct password to access admin tools.
  • Admins can view, delete, or manage all appointments.
  • To reset appointment IDs after deleting all records, use the database command:
    ALTER TABLE appointments AUTO_INCREMENT = 1;
